### 版本

- jdk 1.7

- spring 4.2.4 

- mybatis 3.2.8 

- 开发工具：eclipse

- 数据库：mysql


### 启动

Simple-SSM 项目右键->Debug  As  -> Maven build ...  

Goals ： tomcat7:run 

 
### 浏览器访问

http://localhost:8080/Simple-SSM/test
